# Pathfinding Java (Simple Layout)

This is a simple-layout Java port of the Pathfinding project. 
Put the `src/` folder in a Java IDE or compile with `javac` and run `Main`.

Run example (from project root):
```
javac src/*.java
java -cp src Main --map maps/small.txt --algo astar --start 0,0 --goal 4,4 --diag false --log runlog.json
```
